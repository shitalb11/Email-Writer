**EMAIL WRITER: This project helps users generate professional emails using AI.**
An AI-powered Email Writer project that includes:
• Frontend Application
• Backend API
•Chrome Extension

**Project Structure**
Email-Writer/
│
├── Backend/      # Spring Boot Backend
├── Frontend/     # React Frontend
└── Extension/    # Chrome Extension

**Technologies Used**
1) Frontend
• React.js
• JavaScript
• Tailwind CSS
2) Backend
• Spring Boot
• Java
• REST API
3) Extension
• Chrome Extension APIs
• JavaScript

**Features**
• AI-generated email responses
• Chrome extension integration
• Modern frontend UI
• Backend API support
• Real-time email generation

**Installation**
1) Clone Repository
• git clone https://github.com/shitalb11/Email-Writer.git
2) Frontend Setup
• cd Frontend
• npm install
• npm run dev
3) Backend Setup
• Open Backend folder in IntelliJ IDEA or STS.
• Run the Spring Boot application.
4) Extension Setup
• Open Chrome
• Go to:
chrome://extensions
• Enable Developer Mode
• Click "Load unpacked"
• Select Extension folder
